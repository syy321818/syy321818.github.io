VBA Code to Send Email from Excel (Outlook Automation)

The Problem

Sending the same email to multiple people with different attachments is tedious. Doing it manually usually leads to mistakes (like attaching the wrong file or forgetting the subject line).

The Solution

We can use VBA to “talk” to Outlook. This script creates a new email, fills in the recipient, subject, and body, attaches a specific file, and displays it for you to review (or sends it immediately).

Note: You must have Microsoft Outlook installed and logged in.

How to Customize It
