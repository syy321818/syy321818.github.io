How to Fix Runtime Error 9 (Subscript out of range)

Introduction

“Runtime Error 9: Subscript out of range” occurs when your code attempts to access a member of a collection or an array that does not exist. Think of it as asking for “Drawer 5” when the cabinet only has 4 drawers.

Common Causes

  1. Sheet Name Spelling: The tab name in Excel doesn’t match the code exactly (e.g., “Sheet1 “ vs “Sheet1”).
  2. Array Limits: Trying to access index 5 in an array that only goes up to 3.
  3. Closed Workbooks: Trying to reference a workbook that isn’t open.
