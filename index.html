VBA Script to Save Active Sheet as PDF

The Script

This macro saves the current sheet as a PDF in the same folder as the Excel file.

Sub SavePDF()
    Dim PDFName As String
    
    If ThisWorkbook.Path = "" Then
        MsgBox "Save file first!"
        Exit Sub
    End If
    
    PDFName = ThisWorkbook.Path & "\" & ActiveSheet.Name & ".pdf"
    
    ActiveSheet.ExportAsFixedFormat Type:=xlTypePDF, Filename:=PDFName
    
    MsgBox "PDF Saved: " & PDFName
End Sub

How to Merge Multiple Excel Files into One (VBA Script)

The Script

This script loops through a folder and combines all Excel files into your current workbook.

Sub MergeFiles()
    Dim FolderPath As String, Filename As String
    Dim Sheet As Worksheet, LastRow As Long
    Dim Master As Workbook, Source As Workbook
    
    ' CHANGE THIS PATH (Keep the slash at the end)
    FolderPath = "C:\Users\YourName\Documents\Files\"
    
    Set Master = ThisWorkbook
    Filename = Dir(FolderPath & "*.xlsx")
    
    Do While Filename <> ""
        Set Source = Workbooks.Open(FolderPath & Filename)
        LastRow = Source.Sheets(1).Cells(Rows.Count, 1).End(xlUp).Row
        
        If LastRow > 1 Then
            Source.Sheets(1).Range("A2:Z" & LastRow).Copy
            Master.Sheets(1).Range("A" & Rows.Count).End(xlUp).Offset(1, 0).PasteSpecial xlPasteValues
        End If
        
        Source.Close False
        Filename = Dir
    Loop
    MsgBox "Done!"
End Sub

How to Fix Runtime Error 9 (Subscript out of range)

“Runtime Error 9: Subscript out of range” occurs when your code attempts to access a member of a collection or an array that does not exist. Think of it as asking for “Drawer 5” when the cabinet only has 4 drawers.

  1. Sheet Name Spelling: The tab name in Excel doesn’t match the code exactly (e.g., “Sheet1 “ vs “Sheet1”).
  2. Array Limits: Trying to access index 5 in an array that only goes up to 3.
