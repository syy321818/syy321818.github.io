VBA Script to Save Active Sheet as PDF

Posted on 2025-12-09 In Automation Scripts

The Script

This macro saves the current sheet as a PDF in the same folder as the Excel file.

Sub SavePDF()
    Dim PDFName As String
    
    If ThisWorkbook.Path = "" Then
        MsgBox "Save file first!"
        Exit Sub
    End If
    
    PDFName = ThisWorkbook.Path & "\" & ActiveSheet.Name & ".pdf"
    
    ActiveSheet.ExportAsFixedFormat Type:=xlTypePDF, Filename:=PDFName
    
    MsgBox "PDF Saved: " & PDFName
End Sub
